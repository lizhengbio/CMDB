# CMDB DEMO#
----------
## 部署说明 ##

- 环境python 2.7.13 django 1.8.2 
- 在命令行运行以下代码  
`git clone git@github.com:lizhengbio/CMDB.git`  
`cd CMDB`  
`pip install -r requirements.txt`  
`python manage.py runserver 0.0.0.0:8000` ##启动服务
